### INSTRUCCIONES

Tienes el siguiente objeto:
const ordenador1 = {
marca: "HP",
procesador: "i7",
ram: "16Gb",
hd: "1Tb",
precio: 1500,
extras: ["cámara HD", "microfono stereo", "disco de acceso rápido"],
maletin: true;
}
- Escribe en consola una frase que diga "mi ordenador es un HP, con procesador i7 y 16Gb de RAM"

- Muestra en consola el tercer extra del ordenador

JSON Pokemon 
Tienes un objeto pokemon con estas características:
{"name": "Bulbasaur", "hp": 45, "attack": 49, "defense": 49, "sp_atk": 65, "sp_def": 65, "speed": 45, "types": ["poison", "grass"], "abilities": ["overgrow", "chlorophyll"], "evolutions": ["Ivysaur"], "description": "Durante un tiempo después de nacer, crece alimentándose de la semilla que lleva en su espalda."}
1- ¿Cuál es el valor de ataque (attack) de Bulbasaur?
2 - ¿Qué tipos tiene Bulbasaur?
3 - ¿Cuáles son las habilidades (abilities) de Bulbasaur?
4 - ¿A qué Pokémon evoluciona Bulbasaur?
5 - ¿Cuál es la descripción de Bulbasaur en el JSON?

6- Usar Object.keys(), Object.values() y Object.entries() para mostrar claves, valores y ambos respectivamente

7- ¿Cómo mostrarías todos estos resultados en pantalla?
